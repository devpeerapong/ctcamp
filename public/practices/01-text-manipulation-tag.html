เท่านั้น ถึงจะมีการเว้นบรรทัดได้

เพราะว่า Web Browser จะทำการมองการช่องว่างต่าง ๆ เป็นแค่การเว้นวรรคหนึ่งครั้งเท่านั้น ทดสอบได้ด้วยการ เว้น วรรคยาว ๆ ในไฟล์ HTML

และ TAG br ถือเป็น TAG พิเศษ ที่ไม่ต้องมี TAG ปิด เนื่องจาก TAG นี้ไม่ได้เน้นข้อความ แต่เป็น TAG ที่มีไว้เพื่อบอกว่า จุดนี้มีไว้เพื่อเว้นบรรทัดเท่านั้น

ข้อความต่าง ๆ ก็สามารถอยู่ได้โดยไม่มี TAG ครอบเช่นกัน แต่เราควรจะใช้ประโยชน์จาก TAG ต่าง ๆ มากกว่า

นอกจาก TAG สำหรับหัวข้อกับข้อความแล้ว ยังมี TAG ช่วยต่าง ๆ เช่น TAG strong ที่จะทำให้ ข้อความที่ถูกครอบกลายเป็นตัวหนา
